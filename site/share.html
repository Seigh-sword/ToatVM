Sharing your VM

Every ToatVM session gets a public toatvm-<random>.trycloudflare.com URL. Anyone with the link can open it in a browser. To control who gets in, set a share password — the tunnel then requires HTTP basic auth (user toat + your password).

Set a password

toatvm -share            # prompts for a password
toatvm -share hunter2   # or pass it directly

This restarts the current session so ttyd picks up the password. Share the URL; people connect with user toat and your password.

Remove the password

toatvm -unshare

Upload files to share

toatvm -sync ./my-folder   # uploads into $HOME/sync on the VM

-sync tars your folder, base64-encodes it, and streams it into the VM over the same tunnel — no extra tooling required.

Even with a password, the tunnel is still a public endpoint. Share URLs only with people you trust, and don't leave sensitive data on a shared VM.
