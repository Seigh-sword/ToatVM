Install & Download

npm (recommended)

npm i -g toatvm
toatvm -help

Requires Node.js 18 or newer.

npx (no install)

npx toatvm -init

From source

git clone https://github.com/Seigh-sword/ToatVM.git
cd ToatVM/cli
npm install
npm run build
node dist/index.js -help

Windows 10 / 11

The CLI runs natively on Windows. After booting, use toatvm -open to launch the URL in your default browser, or toatvm -ssh to get a command you can paste into Windows Terminal / PowerShell.

First run

toatvm -new     # create an account (owner / repo / PAT)
toatvm -init    # interactive wizard: OS, boot, controls

The Personal Access Token needs repo and workflow scopes. It is stored only in ~/.config/toatvm/config.json on your machine.
